Transportadora PHP7

Esse foi um projeto desenvolvido no curso técnico de informática no primeiro semestre de 2018.
O objetivo deste, é simular um sistema de Transportadora, o foco do projeto não é o frontend!

Podemos cadastrar motoristas e caminhões, assim como deletá-los e alterá-los. O projeto tem também a
padronização e a verificação dos dados cadastrados.

------------------------------------------------------------------------------------------------------

A IDE usada foi o Atom. E para testes locais xampp.
